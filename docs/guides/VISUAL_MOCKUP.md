## Key Visual Features

### 1. **Market Bias Banner**
- Full width
- Bold gradient (green for bullish, red for bearish)
- 20px font, 800 weight
- Impossible to miss

### 2. **Command Center Boxes**
- 4 equal-sized boxes in a row
- Each 150px tall
- Gradient backgrounds:
  - Price: Blue/cyan or red/pink (depending on sentiment)
  - Resistance: Orange/pink gradient
  - Support: Dark blue/purple gradient
  - Flip: Teal/pink gradient
- White text, 36px values
- Hover effect: Lifts up 5px with darker shadow

### 3. **Alerts Section**
- Compact, 3 alerts max
- Color-coded borders (red/orange/green)
- Inline action with arrow (â†’)
- Easy to scan

### 4. **2x2 Chart Grid**
- Equal-sized quadrants
- 400px height each (compact but readable)
- All visible without scrolling
- Clear labels above each chart

### 5. **Expandable Sections**
- Chart guide collapses by default
- Multi-expiry below charts
- Educational content at bottom
- Keeps main view clean
